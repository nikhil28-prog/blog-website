# Simple Blog Website (Flask + SQLite)

This is a simple Blog Website built using Python and Flask.
Users can add, view, edit, and delete blog posts.
All posts are stored in a local SQLite database.

## Features

* Add new blog posts
* View all posts
* Edit existing posts
* Delete posts
* Data stored in SQLite database
* Simple and clean user interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS

## How to Run the Project

1. Install Flask:

pip install flask

2. Run the application:

python blog_app.py

3. Open your browser and go to:

http://127.0.0.1:5000
